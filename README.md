# Geet Patil Portfolio

Hudson-inspired Astro portfolio for Geet Patil.

Website:

```text
https://kirmada102.github.io/Geet-Patil/
```

Development:

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
```

GitHub Pages deploys the generated `dist/` folder through `.github/workflows/pages.yml`.
